# Swit-her для macOS

Простой аналог Punto Switcher без автопереключения. Нажми **Option (Alt)** или **Ctrl** — и последнее слово (или выделенный текст) переключится между русской и английской раскладками.

Ваши данные никуда не улетают и программа не заставляет скачивать ненужный браузер.


---

## Установка

### Быстрый способ (DMG)

1. Скачай `dist/Swit-her.dmg` либо найди файл Swit-her.dmg в Releases 1.0.0
2. Открой образ: `open dist/Swit-her.dmg`
3. Перетащи `Swit-her.app` в папку Applications
4. Выдай разрешение Accessibility (см. ниже)
5. Запусти приложение

### Из исходников

```bash
chmod +x install.sh && ./install.sh
./build.sh
```

После сборки в `dist/` появятся:
- `Swit-her.app` — приложение
- `Swit-her.dmg` — установочный образ

### Разрешение Accessibility (обязательно!)

Без этого приложение не сможет перехватывать клавиатуру и работать с текстом.

**Системные настройки → Конфиденциальность и безопасность → Универсальный доступ**
→ нажми `+` → добавь `Swit-her.app`

После добавления — перезапусти приложение.

## Использование

| Действие | Результат |
|----------|-----------|
| Нажать **Option** или **Ctrl** (без выделения) | Переключает раскладку **последнего слова** |
| Выделить текст → нажать **Option** или **Ctrl** | Переключает раскладку **выделенного текста** |
| Клик по иконке в меню-баре | Открывает меню с настройками |
| Меню → Переключить сейчас | Переключает раскладку выделенного текста |
| Меню → Клавиша переключения | Выбор между Alt и Ctrl |

Работает в любом приложении: браузер, TextEdit, Telegram, Finder, IDE и т.д.

## Автозапуск при входе в систему

1. Открой файл `com.user.swit-her.plist`
2. Замени `ТВОЙ_ЮЗЕР` на своё имя пользователя (узнать: `whoami` в терминале)
3. Скопируй в LaunchAgents:

```bash
cp com.user.swit-her.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.user.swit-her.plist
```

Для отключения автозапуска:
```bash
launchctl unload ~/Library/LaunchAgents/com.user.swit-her.plist
```

ЛИБО: Настройки → Основные → Объекты входа → Добавить приложение

## Как это работает

1. Приложение создаёт **CGEventTap** — системный перехватчик событий клавиатуры macOS (Quartz API).
2. При нажатии **чистого Option/Ctrl** (без Cmd/Shift и других модификаторов):
   - Если есть выделение — копирует его через `Cmd+C`
   - Если нет — выделяет последнее слово через `Option+Shift+Left`
   - Определяет раскладку по первому буквенному символу
   - Конвертирует через таблицу перекодировки (позиции клавиш ru↔en)
   - Вставляет результат через `Cmd+V`
   - Переключает системную раскладку на соответствующий язык
3. Модификатор **не проходит** в приложение, только если нажат без других клавиш.
4. Выбор клавиши (Alt/Ctrl) сохраняется в NSUserDefaults и восстанавливается при запуске.

## Ограничения

- **Не работает** в полноэкранных играх и приложениях, блокирующих Accessibility API
- В некоторых приложениях (например, защищённых паролем полях) `Cmd+C` может не сработать
- Приложение занимает ~0% CPU в режиме ожидания (CFRunLoop спит до события)

---

## Требования

- macOS 11+ (Big Sur и новее)
- Python 3.9+
- rumps, pyobjc (устанавливаются автоматически через `install.sh`)
