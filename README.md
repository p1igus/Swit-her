# Layout Switcher для macOS

Простой аналог Punto Switcher. Нажми **Option (Alt)** — и последнее слово (или выделенный текст) переключится между русской и английской раскладками.

---

## Установка

### 1. Распакуй файлы

Положи папку `layout_switcher/` куда удобно, например:
```
~/layout_switcher/
```

### 2. Установи зависимости

```bash
cd ~/layout_switcher
chmod +x install.sh
./install.sh
```

Или вручную:
```bash
pip3 install pyobjc-core pyobjc-framework-Cocoa pyobjc-framework-Quartz
```

### 3. Выдай разрешение Accessibility (обязательно!)

Без этого скрипт не сможет перехватывать клавиатуру и работать с текстом.

**Системные настройки → Конфиденциальность и безопасность → Универсальный доступ**
→ нажми `+` → добавь `Terminal.app` (или iTerm2 / Python)

После добавления — перезапусти Terminal и запусти скрипт снова.

### 4. Запуск

```bash
python3 ~/layout_switcher/layout_switcher.py
```

## Использование

| Действие | Результат |
|----------|-----------|
| Нажать **Option** (без выделения) | Переключает раскладку **последнего слова** |
| Выделить текст → нажать **Option** | Переключает раскладку **выделенного текста** |

Работает в любом приложении: браузер, TextEdit, Telegram, Finder, IDE и т.д.

---

## Автозапуск при входе в систему

1. Открой файл `com.user.layout-switcher.plist`
2. Замени `ТВОЙ_ЮЗЕР` на своё имя пользователя (узнать: `whoami` в терминале)
3. Скопируй в LaunchAgents:

```bash
cp com.user.layout-switcher.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.user.layout-switcher.plist
```

Для отключения автозапуска:
```bash
launchctl unload ~/Library/LaunchAgents/com.user.layout-switcher.plist
```

## Как это работает

1. Скрипт создаёт **CGEventTap** — системный перехватчик событий клавиатуры macOS (Quartz API).
2. При нажатии **чистого Option** (без Cmd/Ctrl/Shift):
   - Если есть выделение — копирует его через `Cmd+C`
   - Если нет — выделяет последнее слово через `Option+Shift+Left`
   - Определяет раскладку по первому буквенному символу
   - Конвертирует через таблицу перекодировки (позиции клавиш ru↔en)
   - Вставляет результат через `Cmd+V`
3. Option **блокируется** (не проходит в приложение), только если нажат без модификаторов.

---

## Ограничения

- **Не работает** в полноэкранных играх и приложениях, блокирующих Accessibility API
- В некоторых приложениях (например, защищённых паролем полях) `Cmd+C` может не сработать
- Скрипт занимает ~0% CPU в режиме ожидания (CFRunLoop спит до события)

---

## Требования

- macOS 11+ (Big Sur и новее)
- Python 3.9+
- pyobjc (устанавливается автоматически через `install.sh`)
